# 브라우저 호환성 및 반응형 테스트 체크리스트

## 📱 반응형 브레이크포인트
- **모바일**: ≤ 480px (iPhone SE, 8)
- **태블릿**: 481px - 768px (iPad)
- **데스크톱**: 769px - 1024px (태블릿 가로)
- **큰 화면**: ≥ 1025px (Full HD)

---

## 🔍 테스트할 사항

### 1️⃣ 헤더/네비게이션
- [ ] **데스크톱** (≥1025px): 수평 메뉴 정상 표시
- [ ] **모바일** (≤768px): 햄버거 메뉴 보임
- [ ] **메뉴 클릭**: 페이지 이동 정상
- [ ] **로고**: 모든 화면에서 정상 크기
- [ ] **메뉴 우측 정렬**: "CONTACT" 마지막 항목

### 2️⃣ 콘텐츠 영역
- [ ] **텍스트**: 줄바꿈 자연스러움
- [ ] **이미지**: 화면 너비에 맞게 스케일
- [ ] **여백**: 모바일에서 padding 충분함
- [ ] **가독성**: 글자 크기 충분함

### 3️⃣ 색상 & 시각 요소
- [ ] **배경색**: #f8f9ff (섹션) 정상
- [ ] **그라디언트**: #667eea → #764ba2 정상
- [ ] **아이콘**: Font Awesome 보임
- [ ] **링크 색상**: #667eea (호버 시 #764ba2)

### 4️⃣ 반응형 확인

#### 480px (모바일)
- [ ] 텍스트 폰트 크기: 작아짐 (14-16px)
- [ ] 패딩: 줄어듦 (15-20px)
- [ ] 버튼: 터치 가능한 크기 (최소 44px)
- [ ] 이미지: 전체 너비 사용

#### 768px (태블릿)
- [ ] 메뉴: 아직 모바일 모드
- [ ] 텍스트: 읽기 편한 크기
- [ ] 이미지: 최대 너비 적용

#### 1024px (데스크톱)
- [ ] 메뉴: 수평 표시
- [ ] 최대 너비: 1200px 적용
- [ ] 타이포그래피: 정체
